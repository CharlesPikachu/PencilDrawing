# Environment

	- pillow
	- scipy

# Usage
#### Step1
```sh
Install the neccessary dependencies:
run "pip install -r requirements.txt"
```
#### Step2
```sh
run as "python main.py -i <image_path> -m <mode> -s <savename>", for example:
python main.py -i testPic/1.jpg -m color -s output.jpg
```
